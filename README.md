## Introduction
这是一个微博评论分析工具，实现功能主要有：
1. 微博评论数据爬取；
2. 分词与关键词提取；
3. 词云与词频统计；
4. 情感分析；
5. 主题聚类

实现的效果在目录下: “ 案例：泰国大象踩踏伤人事件 ”

*Codes also works on Twitter and I may update a new repository about it.*
